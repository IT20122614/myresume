Project Description

This is a new online job portal website called GETJOBS.LK. Here a visitor can be either an employer or a job seeker. Visitors can search for vacancies and he can view job vacancies. If the vacancy that visitors looking for is not available anymore system notifies the visitor. Visitors can register for the system. If the visitor is an employer, he/she should enter payment details for the monthly charges while the registration. Then the system validates the payment details. Then visitors get an activation code via an email after visitors can enter that code activate his/her account.

A registered job seeker can log in to the system. Also, he can edit his/her profile details. A registered job seeker can apply for a job that he/she is interested in he/she can view the job description before applying for the job. He/she can fill the application and submit it with a CV. If there are any errors or invalid data, the system gives him/her an error message. Otherwise system display “application submission success”. A registered job seeker can see employer details and he/she can contact them via a call or an email if he/she is having any issue.

A registered employer also can log in to the system and he/she can edit their profile details as well. A registered employer can post an ad and he/she can modify them as well. Before posting an ad he/she must pay for the ad otherwise system won’t allow posting the ad. Moreover, he/she can view the applications that job seekers have submitted. If he/she is satisfied with the job seeker he/she can send a message saying, “your application was accepted”. Then he/she can give an interview conference link (the interview is conducting via live conference) via an email.

The system administrator can approve the job vacancies and employers. Also, he/she can delete a vacancy or an employer from the system. And he/she can give the reason also. He/she is the one who maintains the website. Admin can get income reports through the system. He/she can get it as monthly income report or ad income report.
